## 添加记录

往 `list.txt` 内写入即可。一行一条记录，支持 Steam APPID 与 Steam Store 商店页链接。

点击该链接即可：https://github.com/gamescalendar/gamescalendar.github.io/edit/master/list.txt

## 覆盖记录

在 `override.json` 中的特定字段将会覆盖自动拉取的字段。包括：`start`, `app_data.owned`。

## 静态记录

在 `static.json` 中记录即可。
